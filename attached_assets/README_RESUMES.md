# Resume Files

To enable resume downloads in the portfolio, place your resume PDF files in this directory:

1. **SDE Resume**: Name the file `SDE_Resume.pdf` or `SDE resume.pdf`
2. **AI Resume**: Name the file `AI_Resume.pdf` or `AI_KISHORE_KUMAR S_S Resume.pdf`

The backend will automatically serve these files when users click the "View Resume" buttons in the Hero section or Footer.

## File Naming Options

The backend checks multiple possible file names:

### SDE Resume
- `SDE_Resume.pdf` (recommended)
- `SDE resume.pdf`
- `/mnt/data/SDE resume.pdf` (if available)

### AI Resume
- `AI_Resume.pdf` (recommended)  
- `AI_KISHORE_KUMAR S_S Resume.pdf`
- `/mnt/data/AI_KISHORE_KUMAR S_S Resume.pdf` (if available)
